## Как опубликовать в интернет

Бесплатные варианты:

1. **GitHub Pages** — залить в репозиторий, включить Pages
2. **Netlify** — перетащить папку на [netlify.com/drop](https://app.netlify.com/drop)
3. **Vercel** — `npx vercel`

После публикации получите ссылку вида `https://tamerlan-diana.netlify.app` — её и отправляете гостям.

## Анкета

На сайте — **своя форма** в дизайне приглашения. Ответы гостей приходят на почту `dians.ts.29.09@gmail.com` через [FormSubmit](https://formsubmit.co) — каждый ответ отдельным письмом в виде таблицы.

**Один раз перед запуском:** FormSubmit пришлёт на эту почту письмо «Activate Form» — нужно нажать ссылку, иначе ответы не дойдут.

### Почему не чужая Яндекс.Форма

Ссылка из переписки — пример с другой свадьбы. Встраивать её нельзя: ответы ушли бы чужим людям.

### Если нужна таблица ответов (как у Яндекс.Форм)

| Вариант | Плюсы | Минусы |
|--------|-------|--------|
| **Своя форма + почта** (сейчас) | Красиво, уже готово | Ответы в почте, не в одной таблице |
| **Своя Яндекс.Форма** | Таблица в Яндексе, Ольга сама управляет | iframe выбивается из дизайна сайта |
| **Уведомления в Telegram** | Красиво + мгновенно в чат организатору | Нужно создать бота (15 мин) |

**Рекомендация:** оставить свою форму. Для Ольги и Дианы — смотреть ответы в почте `dians.ts.29.09@gmail.com` (можно дать доступ организатору или настроить пересылку).

Если позже Ольга создаст **свою** Яндекс.Форму с теми же вопросами — можно будет встроить её вместо текущей формы.
